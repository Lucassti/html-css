Citações e Códigos

Código-Fonte / Pré Formatação

Utilizamos o <code> para citar uma parte de Código-Fonte, por exemplo:
num = (int input("Digite um número")) print = num


Fica restando apenas pré formatar para ficar igual citado no código utilizamos <pre>, ficando: 

            

num = (int input("Digite um número"))
print = num

posso utilizar shift + tab e seleciona o texto para diminuir o espaçamento lateral.


Citações Simples

Utilizamos a citação com a tag <q>, assim entregando o significado e não forma, por exemplo:
Como diz o pai de um amigo: o computador é um burro muito rápido.

Citações Completas

Podemos utilizar a citação completa para indicar um site de autor de livro, vendas e outros.
utilizamos a tag <blockquote> e após cite="link de referência">, por exemplo:
Citado no curso em vídeo por Gustavo Guanabara

HTML é uma semântica e CSS forma.


Abreviação

Utilizamos a tag <abbr>, assim você entrega a informação do texto abreviado, por exemplo:
HTML e CSS
